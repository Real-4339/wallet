# Domain Models

## User

```csharp
class User {
    User Create();
    void CreateWallet();
    void Update();
    void UpdateWallet();
    void Delete();
}
```

```csharp
class Wallet {
    Wallet Create();
    void UpdateBalance();
    void AddTransaction();
    decimal GetBalance();
}
```

> Wallet is an Entity, User is an Aggregate Root
